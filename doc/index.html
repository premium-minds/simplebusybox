SimpleBusyBox

SimpleBusyBox is a BusyBox installation for Android ARM. It does not require root. Simply tell SimpleBusyBox to unpack, and then copy and paste the PATH into your shell and you can use BusyBox versions of various utilities.

Quick start

  • Install and launch SimpleBusyBox.
  • Tap "Unpack" to unpack the busybox binary into a directory and build all of appropriate symlinks.
  • SimpleBusyBox -> Menu (upper right) -> Copy PATH
  • Now go to a shell/terminal app and paste the PATH command.
